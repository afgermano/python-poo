# Sistema de Restaurante

Projeto em Python utilizando Programação Orientada a Objetos (POO).

## Funcionalidades

* Cadastro de restaurantes
* Listagem de restaurantes
* Avaliação de restaurantes
* Cálculo da média das notas

## Como executar

```bash
python main.py
```

## Objetivo

Praticar lógica de programação e conceitos de POO.
